### get started
- global로 typescript 설치 (이미 설치시 생략 가능)
    - sudo npm install -g typescript
- tsconfig.json 설치
    - tsc --init
- npm 설치
    - npm init -y
- express 설치
    - npm i express

### 실행
- 컴파일
    - tsc [file_name.ts]
    - tsc -w
