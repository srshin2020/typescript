### get started
- global로 typescript 설치
    - sudo npm install -g typescript
- version check 
    - tsc -v
- 컴파일
    - tsc [file_name.ts]
    - tsc -w
    - tsc -ww [filename]
- tsconfig.json 설치
    - tsc --init
### extention
- prettier
- eslint : code 품질 도구
- path intellisence :import를 도움
- bracket pair colizer : 괄호 쌍에 대해 색으로 매칭
